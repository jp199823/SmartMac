# SmartMac

A native macOS system monitoring application built with SwiftUI that provides real-time system insights, detailed technical specifications, and intelligent optimization suggestions.

## Requirements

- macOS 13.0 or later
- Xcode 15.0+ (for building from source)

## Building from Source

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SmartMac.git
   cd SmartMac
   ```

2. Build using Swift Package Manager:
   ```bash
   swift build -c release
   ```

3. Or open in Xcode and build directly.
